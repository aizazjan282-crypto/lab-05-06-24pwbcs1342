## **Introduction**
This repository contains all tasks for **Lab 05 & 06** on **CSS Layout Techniques and Responsive Design**. The lab focuses on mastering:

- CSS **Display Properties**  
- **Float-based layouts** and clearfix techniques  
- CSS **Positioning** (static, relative, absolute, fixed, sticky)  
- **Flexbox** and **CSS Grid** layouts  
- **Responsive Design** using media queries  
- **Mobile-first development** and utility classes  

Each task is implemented in a **separate HTML and CSS file**, demonstrating different layout techniques and responsive behavior.
